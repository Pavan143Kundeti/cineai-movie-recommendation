QUICK START (5 STEPS)
================================================================================

STEP 1: Create MySQL Database
------------------------------
mysql -u root -p
CREATE DATABASE movie_recommender;
EXIT;

STEP 2: Setup Backend
---------------------
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env

STEP 3: Configure .env File
---------------------------
Edit backend\.env with:
- Your MySQL password
- TMDB API key (get from https://www.themoviedb.org/settings/api)
- OMDB API key (get from http://www.omdbapi.com/apikey.aspx)
- Random SECRET_KEY (32+ characters)

STEP 4: Migrate Data
--------------------
python migrate_data.py

STEP 5: Start Servers
---------------------
Terminal 1 (Backend):
  cd backend
  venv\Scripts\activate
  uvicorn app.main:app --reload

Terminal 2 (Frontend):
  cd frontend
  streamlit run app.py

================================================================================
TROUBLESHOOTING
================================================================================

Problem: Can't connect to database
Solution: Check MySQL is running, verify DATABASE_URL in .env

Problem: Module not found
Solution: Activate venv, run: pip install -r requirements.txt

Problem: API key error
Solution: Get free keys from TMDB and OMDB, add to .env

Problem: Migration fails
Solution: Ensure CSV files exist, check database permissions
